# 🌌 Overview

**CardioVerse AI** is a futuristic AI-powered healthcare application that predicts the risk of heart disease using Machine Learning and an ultra-modern cyberpunk inspired UI.

This project combines:

- 🧠 Artificial Intelligence
- 🫀 Healthcare Analytics
- ⚡ Real-Time Predictions
- 🌌 Futuristic UI/UX
- 🚀 Streamlit + Machine Learning

into one visually stunning experience.

# 🛠️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/CardioVerse-AI.git
```

---

## 2️⃣ Move Into Project Folder

```bash
cd CardioVerse-AI
```

---

## 3️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

# 🚀 Run Application

```bash
streamlit run app.py
```
